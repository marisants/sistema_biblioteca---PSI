# Sistema de Biblioteca
# RELATÓRIO

## Resumo
Este projeto tem como objetivo desenvolver um sistema de biblioteca web utilizando o framework Flask, com a finalidade de facilitar o gerenciamento e a organização de livros. A equipe do projeto é composta por: Arthur Vinicius Medeiros, Mariana Sâmara Azevedo Santos e Wemylly Joice Dutra Saraiva. O sistema será criado para resolver o problema da falta de organização no controle de livros, que muitas vezes é feito de forma manual e pouco eficiente, dificultando a busca por informações, o cadastro de novos livros e a atualização dos dados, além de facilitar esse gerenciamento, evitando a sobrecarga dos funcionários da biblioteca e apresentando uma nova forma de empréstimo de livros, onde o aluno procura na aplicação o livro que deseja e se estiver disponível vai à biblioteca para fazer a reserva.

Dessa forma, o desenvolvimento do sistema se justifica pela necessidade de tornar o gerenciamento da biblioteca mais prático e eficiente, permitindo o cadastro, edição, remoção e visualização dos livros de forma simples e organizada. Além disso, o projeto também é importante para aplicar na prática os conhecimentos adquiridos em sala de aula contribuindo para o aprendizado e desenvolvimento dos integrantes da equipe.

## Descrição Objetiva da Proposta

O projeto consiste no desenvolvimento de um **Sistema de Biblioteca Web**, utilizando o framework Flask, com o objetivo de facilitar o gerenciamento de livros de uma biblioteca. A aplicação permitirá o cadastro de usuários, login e logout, além do gerenciamento completo de livros através de operações de cadastro, edição, exclusão e visualização.

O sistema permitirá que usuários autenticados possam acessar funcionalidades específicas, garantindo maior organização e controle das informações.

## Problema a ser Resolvido

Muitas bibliotecas ainda realizam o controle de livros manualmente ou com sistemas pouco eficientes, dificultando a organização, busca e controle do acervo. Isso pode gerar problemas como :

* Dificuldade na busca por livros
* Falta de organização do acervo
* Perda de informações
* Controle ineficiente dos registros

## Justificativa

O desenvolvimento de um sistema web para biblioteca torna o gerenciamento mais eficiente, organizado e acessível. A aplicação permitirá que os usuários encontrem livros facilmente, além de facilitar o cadastro e controle das informações.

## Funcionalidades do Sistema

* Cadastro de usuários
* Login de usuários
* Logout do sistema
* Cadastrar livro
* Visualizar livros
* Editar livro
* Remover livro
* informar Título
* informar Autor
* informar Ano
* informar Gênero

## Tecnologias Utilizadas

* Python
* Flask
* HTML
* CSS
* Jinja2 (templates)
* JSON

## Conclusão

O sistema de biblioteca permitirá o gerenciamento eficiente dos livros, além de aplicar na prática os conceitos estudados sobre Flask. O projeto contribui para o aprendizado dos integrantes e resolve um problema real de organização e controle de bibliotecas.
